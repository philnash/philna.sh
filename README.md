# philna.sh

This repository is the source code of the personal site and blog of [Phil Nash](https://philna.sh).

It is powered by [Astro](https://astro.build/) and uses:

- [@philnash/astro-pagination](https://github.com/philnash/astro-components/tree/main/packages/astro-pagination) for accessible pagination
- [@philnash/astro-related-content](https://github.com/philnash/astro-components/blob/main/packages/astro-related-content) to suggest related links to blog posts
