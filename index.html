About

Phil Nash is a developer evangelist. Sometimes he writes code on stage in front of a crowd, hoping everything just works. Sometimes he writes open source code, which is much less stressful because if it is wrong someone else can correct it. He writes code in tweets sometimes, but not much fits. He live codes on Twitch. He blogs on this site and on the Twilio blog and you'll find plenty of code there too.
